## Description

social network for blogs

## Installation

```bash
$ npm install
```

## Configuration

- Copy the [`.env.example`](./.env.example) file and call it `.env`.

- Edit the environment variables in the .env file as you need.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
